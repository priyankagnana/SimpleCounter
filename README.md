# Simple Counter

A minimal React counter app with increment, decrement, and reset.

## Tech Stack

- React 18
- Vite

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the URL shown in the terminal (e.g. http://localhost:5173).

## Build

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Usage

- **+** — increase the count  
- **−** — decrease the count  
- **Reset** — set the count back to 0
